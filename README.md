# EventHub - Comprehensive Event Management System

EventHub is a powerful, full-stack platform designed for organizing, discovering, and booking events. It consists of a high-performance Next.js frontend and a robust Node.js/GraphQL backend.

## 📁 Project Structure

- `frontend/`: Next.js application with Ant Design and Framer Motion.
- `backend/`: Node.js server with Express, GraphQL, and MongoDB.

## 🚀 Quick Start

### 1. Backend Setup
Navigate to the `backend` directory and follow the instructions in its README.
```bash
cd backend
npm install
npm run dev
```

### 2. Frontend Setup
Navigate to the `frontend` directory and follow the instructions in its README.
```bash
cd frontend
npm install
npm run dev
```

## 🛠️ Main Features

- **Organizers**: Event management, revenue analytics, attendee roster exports, and QR-based check-ins.
- **Attendees**: Event discovery, Stripe-integrated booking, digital passes, and an AI-powered assistant.
- **Super Admin**: Platform-wide monitoring and support ticket management.
- **AI Integration**: Custom chatbot for user guidance and event queries.

## 🔧 Prerequisites

- **Node.js**: v18.0.0 or higher
- **MongoDB**: For data persistence
- **Stripe Account**: For payment processing
- **Cloudinary**: For image hosting
